    알고리즘 조회

    생성 버튼을 클릭하여 사용할 데이터를 가져온다.

    등록방법

    데이터의 열별 그래프를 확인하고 전처리를 클릭한다. 예시로 사용된 Train 데이터는 코스타리카 가정의 빈곤 수준을 예측하기 위해 80가지의 Features를 가지고 가정의 소득 계층(target)을 4가지 분류로 나눈 데이터이다.
